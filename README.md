# calculadora_de_determinantes
Esta é uma calculadora simples de determinantes de matrizes 2x2, 3x3 e sistemas lineares de até 3 incógnitas, para ter um melhor caminho para atualizações utilizamos o método de Laplace, pois nos permite reutilizar funções propriamente definidas.
Para melhorias futuras é de extrema importância proporcionarmos recursos para formatação do código html, limpeza do código em javascript e mais funções matemáticas como log e raízes
